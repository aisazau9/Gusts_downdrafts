# Gusts_downdrafts
Software for "Super-scaling of wind gusts with thermal downdrafts in cloud-resolving simulations"  

This repository contains code used for the research study "Super-scaling of wind gusts with thermal downdrafts in cloud-resolving simulations" by A. Isaza, S. Sherwood, J. Evans, M. El Rafei, F. Jei, and A. Dowdy
